🔌 Conectar Supabase
Para que los perfiles, fotos y datos de usuarios se guarden en la nube, necesitás configurar Supabase una sola vez.

Pasos rápidos:
1. Entrá a supabase.com y creá un proyecto
2. En Settings → API copiá tu URL y Anon Key
3. Pegálos acá abajo
Verificá que la URL y key sean correctas
Los datos se guardan en tu propia cuenta de Supabase. Gratuito hasta 500MB de storage y 50.000 usuarios.
